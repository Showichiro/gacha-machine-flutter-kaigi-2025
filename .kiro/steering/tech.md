# Technology Stack

## アーキテクチャ

フロントエンドのみのシングルページアプリケーション (SPA)。バックエンドは不要で、すべてのデータはブラウザのローカルストレージに保存されます。

## フロントエンド

### コアフレームワーク
- **Svelte 5.39.6**: リアクティブUIフレームワーク（最新版）
  - 軽量で高速なレンダリング
  - Runes APIによる状態管理
  - コンパイラベースのアプローチ
- **TypeScript 5.9.3**: 型安全な開発環境

### ビルドツール
- **Vite (rolldown-vite 7.1.14)**: 高速なビルドツール
  - 開発サーバーのHMR（Hot Module Replacement）
  - 最適化されたプロダクションビルド
  - Rolldownを使用した高速ビルド

### オーディオライブラリ
- **Howler.js 2.2.4**: サウンド効果とBGMの再生
  - クロスブラウザ対応
  - 複数の音声フォーマットサポート
  - 音量制御とフェード機能

### テストフレームワーク
- **Vitest 4.0.7**: 高速なユニットテスト・統合テスト
- **@testing-library/svelte 5.2.8**: Svelteコンポーネントのテスト
- **Playwright 1.56.1**: E2Eブラウザテスト
- **happy-dom 20.0.10**: 軽量なDOM環境

## 開発環境

### 必須ツール
- **Node.js 22.21.1**: Voltaで管理
- **npm**: パッケージ管理

### 推奨IDE
- **VS Code**: Svelte拡張機能と連携

### コード品質ツール
- **svelte-check**: Svelte固有の型チェック
- **TypeScript compiler**: 静的型チェック

## 一般的なコマンド

```bash
# 開発サーバー起動
npm run dev

# プロダクションビルド
npm run build

# ビルドプレビュー
npm run preview

# 型チェック実行
npm run check

# テスト実行
npm run test

# テストUI起動
npm run test:ui
```

## 環境変数

現時点では環境変数は不要です。すべての設定はローカルストレージで管理されます。

## ポート設定

- **開発サーバー**: デフォルトは5173 (Vite標準)
- **プレビューサーバー**: デフォルトは4173

## データストレージ

- **ローカルストレージ**: ブラウザのlocalStorage APIを使用
  - 景品データの永続化
  - 在庫数の保存
  - アプリ設定の保存

## デプロイメント

静的ファイルホスティングで動作可能:
- Netlify
- Vercel
- GitHub Pages
- その他の静的ホスティングサービス
